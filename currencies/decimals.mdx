### Overview
Setting the number of decimal places for your currency transactions helps maintain consistency and precision within your Discord server's economy. You can set the decimal limit anywhere between 0 and 8.

### How to Set Decimal Places

To set the decimal limit for a currency, you need to access the `/curr menu` for the specific currency ticker.

#### Accessing the Decimals Menu
1. Use the command `/curr menu [ticker]`, where `[ticker]` is the unique identifier of the currency (e.g., GOLD, SILV).
2. In the menu that appears, locate the **Decimals** button highlighted in the red rectangle in the image below:

   ![Decimals Button](https://imgur.com/vImhHpw.png)

#### Selecting Decimal Places
1. Click on the **Decimals** button to open the decimal places selection menu.
2. Choose the desired number of decimal places (between 0 and 8) from the dropdown menu, as shown in the image below:

   ![Select Decimal Places](https://imgur.com/MTgG0Xb.png)

This setting will limit the number of decimal places a user can use when sending the currency to another user.
